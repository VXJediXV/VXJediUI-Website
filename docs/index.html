What You Need

Required & recommended addons.

ElvUI Required

The foundation VXJediUI is built on. Must be installed and kept up to date.

Download → tukui.org
Platynator Recommended

Primary nameplate addon. Handles enemy and friendly nameplate styling.

Download → CurseForge
DandersFrames Recommended

Party and raid frames with click-casting and per-spec profile switching.

Download → CurseForge
Details! Recommended

Damage meter. VXJediUI ships with a custom Details profile tuned for the layout.

Download → CurseForge
BigWigs Recommended

Boss mods for raid and dungeon encounters. Profile pre-configured for the UI layout.

Download → CurseForge
WarpDeplete Recommended

Mythic+ timer with a clean, minimal display that fits the VXJediUI aesthetic.

Download → CurseForge
AyijeCDM Recommended

Cooldown manager with per-spec profiles. Works alongside Blizzard's native CDM.

Download → CurseForge
BuffReminders Recommended

Tracks missing buffs and reminds you to apply them before pulls.

Download → CurseForge
VXJediEssentials Recommended

VXJediXV's own QoL addon — quality of life tweaks bundled with the UI pack.

Included in download

The VXJediUI installer handles all of the above automatically — it detects which addons are loaded and applies the matching profiles. Only ElvUI is strictly required to run VXJediUI. Everything else is optional but recommended for the full experience.

Getting Started

Install in three steps.

01
Install ElvUI first
VXJediUI requires ElvUI — download the latest version from tukui.org and install it before proceeding. Always keep ElvUI up to date for full compatibility.
02
Download & extract VXJediUI
Download the .zip above and unzip it into WoW/_retail_/Interface/AddOns/. Make sure the folder is named VXJediUI exactly.
03
Run the installer in-game
Log in on any character. The setup wizard launches automatically on first login — pick your role, choose a layout, and hit Apply. Done.
